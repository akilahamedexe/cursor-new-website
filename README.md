# SWYM agency website

This repo contains a React + Vite website for **SWYM**, a Dubai-based creative/digital marketing agency (under **AST Solutions**).

## Run locally

From the repo root:

- `cd swym`
- `npm install`
- `cp .env.example .env.local` (optional, for contact form; then fill `VITE_WEB3FORMS_ACCESS_KEY`)
- `npm run dev`

## Build

- `cd swym`
- `npm run build`
