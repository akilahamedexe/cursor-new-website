# SWYM Agency Website

Premium one-page creative/digital marketing website for **SWYM (Dubai, UAE)**, inspired by modern agency layouts with:

- Bold hero section and dark visual system
- Services, work, process, and testimonials sections
- Interactive testimonial slider
- Responsive mobile navigation
- Contact form with client-side success feedback

## Run locally

No dependencies are required.

1. Start a local server:
   - `python3 -m http.server 4173`
2. Open:
   - `http://127.0.0.1:4173`

## Project files

- `index.html` — full website structure/content
- `styles.css` — design system, layout, responsive styles, animations
- `script.js` — interactivity (menu toggle, reveal animation, counters, slider, contact form UX)
