# SWYM — agency website (Vite + React + Tailwind)

Premium, conversion-focused digital marketing website for **SWYM** (Dubai, UAE), under **AST Solutions**.

## Develop

- `npm install`
- `npm run dev`

## Contact form (Web3Forms)

Create `.env.local`:

- `VITE_WEB3FORMS_ACCESS_KEY=...`

## Quality checks

- `npm run lint`
- `npm run build`
